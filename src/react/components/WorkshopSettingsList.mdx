# WorkshopSettingsList

A labelled list of key–value setting rows used in the Workshop Settings page. Each
row shows a setting name on the left and a value (or an editable control) on the right,
separated by a `<Divider>`.

## When to use

- In settings pages where a vertical list of named options needs a consistent label/value
  layout.

## When not to use

- For action lists — use MUI `<List>` with `ListItemButton`.
- For data tables with sortable columns — use MUI `<Table>`.

## Do / Don't

**Do:** group related settings under a section heading so users can scan quickly.

**Don't:** mix settings that have immediate effect with settings that require a Save action
in the same list without clear visual differentiation.
