# FilterChipRow

A horizontally scrollable row of MUI `Chip` filter buttons. The active chip uses
`variant="filled"` with `color="primary"`; inactive chips use `variant="outlined"`.
An empty-string `value` represents "no filter / show all".

## When to use

- For filtering a list by a set of discrete named categories (e.g. lead status, stage).
- When the set of options is small enough to fit in a single scrollable row.

## When not to use

- When the filter options are numerous or hierarchical — use a `<Select>` or
  `PageFilterBar` with a `StageTabGroup` instead.
- When the filter operates across multiple dimensions simultaneously — use a full filter
  panel (popover or drawer) rather than chips.

## Do / Don't

**Do:** always include an "All" chip (key `""`, label `"All"`) as the first option so
users can clear the filter without hunting for a "Clear" button.

**Don't:** show options that have zero matching results — hide or disable empty
chips so users only see filters that will return something.
